# 💰 Finance Goal Tracker PRO

A modern finance tracking application that helps users create savings goals, monitor progress and stay motivated while achieving financial targets.

---

## ✨ Features

- Create multiple finance goals
- Set target amount and saved amount
- Category filtering
- Search goals instantly
- Featured goal section
- Progress bars and statistics
- Total savings overview
- Average completion percentage
- LocalStorage persistence
- Responsive design

---

🔗 Live Demo: https://johnyisbackk.github.io/js-finance-tracker/

---

## 🛠 Built With

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API

---

## 📊 Dashboard Statistics

- Total Saved
- Total Target
- Average Progress
- Featured Goal

---

## 🔎 Goal Management

Users can:

- Add new goals
- Delete goals
- Add money to existing goals
- Search goals
- Filter by category

---

## 💾 Data Persistence

All data is stored inside LocalStorage, allowing users to keep their finance goals even after refreshing the page.

The application is fully responsive and optimized for:
